Specification overview

The ethdebug format is currently in the design phase and specification is only just beginning. Materials related to this effort are available in this section of this website.

Please see  jtoman's initial prototype for a natural-language description of a format that encompasses some of our group's early ideas. See the Schema section for the work-in-progress formal-language schema (using JSON Schema draft 2020-12).
